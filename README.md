# Python NLP Projects Portfolio

This repository contains beginner-level Python projects focused on Natural Language Processing (NLP) and text analysis. It demonstrates fundamental skills in text preprocessing, rule-based analysis, and basic NLP pipeline construction using Python.

## Projects Included

### 1. Keyword-Based Text Analyzer
A simple text analysis tool that processes user input, performs tokenization, calculates word frequency, identifies the most common word, and extracts predefined keywords such as AI, NLP, Python, and data-related terms.

### 2. NLP Emotion Detection (Rule-Based)
A basic emotion detection system that analyzes input text and classifies sentiment using rule-based logic and keyword scoring to determine emotional tone.

## Skills Demonstrated
- Python programming fundamentals  
- Text preprocessing and tokenization  
- Dictionary-based frequency analysis  
- Rule-based NLP systems  
- Basic logic building for text classification  


## Note
These projects are part of early-stage NLP learning and demonstrate foundational concepts in text processing and analysis.
